# Pomodoro Gamificato

App web per la tecnica del Pomodoro con elementi di gamification  

# Funzionalit√† principali

- Timer Pomodoro (25 minuti lavoro, 5 minuti pausa)  
- Gestione punti e badge per ogni pomodoro completato  
- Barra dei progressi che si riempie man mano  
- Salvataggio dei dati in localStorage (punti e pomodori completati persistono anche dopo il refresh)  
- Grafico settimanale dei pomodori completati (Chart.js)  
- Badge animati quando si sbloccano traguardi  


## Interfaccia

- UI semplice e pulita  
- Animazioni leggere per i badge e barra dei progressi  

## Tecnologie utilizzate

- HTML5  
- CSS3  
- JavaScript (ES6)  
- [Chart.js](https://www.chartjs.org/) per il grafico settimanale
